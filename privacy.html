Política de Privacidad

Resumen: CiFlow respeta completamente tu privacidad. Todos tus datos comerciales se almacenan únicamente en tu dispositivo. No recopilamos, enviamos ni compartimos información personal identificable.

1. Información que Recopilamos

CiFlow NO recopila ningún tipo de información personal. La aplicación funciona completamente offline y todos los datos que ingreses se almacenan únicamente en tu dispositivo móvil.

Los datos que TÚ creas y almacenas localmente incluyen:

  • Información de productos (nombres, precios, fotos)
  • Datos de ventas y transacciones
  • Configuración de tu negocio
  • PIN de seguridad (encriptado)
  • Categorías y reportes

2. Cómo Usamos tu Información

Toda la información que ingreses en CiFlow se utiliza exclusivamente para:

  • Proporcionar las funcionalidades de punto de venta
  • Gestionar tu inventario
  • Generar reportes de tu negocio
  • Mantener la seguridad de tus datos

Ninguna información sale de tu dispositivo a menos que TÚ decidas exportarla manualmente.

3. Almacenamiento de Datos

Almacenamiento Local: Todos los datos se guardan en el almacenamiento interno de tu dispositivo Android usando tecnologías seguras como SharedPreferences y almacenamiento de archivos local.

Ubicaciones de almacenamiento:

  • Datos de la aplicación: Almacenamiento interno seguro
  • Imágenes de productos: Directorio privado de la aplicación
  • Backups: Solo cuando TÚ los exportas manualmente

4. Compartir Información

CiFlow NO comparte, vende, alquila ni transfiere tu información personal porque:

  • No tenemos servidores externos para almacenar datos personales
  • La funcionalidad principal no requiere conexión a internet
  • No hay servicios de terceros que accedan a tus datos comerciales
  • Mantenemos el control de tus datos en tu dispositivo

5. Seguridad

Implementamos múltiples medidas de seguridad:

  • Autenticación PIN: Protege el acceso a la aplicación
  • Encriptación: Los datos sensibles están encriptados
  • Almacenamiento local: Sin transmisión externa de datos
  • Control total: Tú decides qué datos exportar o eliminar

6. Permisos de la Aplicación

CiFlow solicita los siguientes permisos únicamente para funcionar:

  • Cámara: Para tomar fotos de productos y escanear códigos de barras
  • Almacenamiento: Para guardar imágenes de productos y exportar backups
  • Internet: Para futuras actualizaciones y funcionalidades opcionales

Estos permisos se usan exclusivamente para las funciones descritas y nunca para recopilar información personal sin tu consentimiento.

7. Control de tus Datos

Tienes control total sobre tus datos:

  • Exportar: Puedes exportar todos tus datos en cualquier momento
  • Eliminar: Puedes borrar todos los datos desde la configuración
  • Transferir: Puedes mover tus datos a otro dispositivo
  • Respaldar: Puedes crear backups cuando quieras

8. Publicidad

CiFlow puede mostrar anuncios publicitarios para mantener la aplicación gratuita. En caso de implementar publicidad:

  • Proveedor: Utilizamos Google AdMob, que puede recopilar datos para personalizar anuncios
  • Datos publicitarios: Los proveedores de anuncios pueden acceder a identificadores de dispositivo para mostrar anuncios relevantes
  • Control: Puedes configurar tus preferencias de anuncios en la configuración de Google
  • Tus datos comerciales: La información de tu negocio (productos, ventas, etc.) NUNCA se comparte con redes publicitarias

Importante: Los anuncios son independientes de los datos de tu negocio y no tienen acceso a tu información comercial.

9. Menores de Edad

CiFlow es seguro para todas las edades ya que no recopila información personal identificable de ningún usuario, incluyendo menores de 13 años. En caso de mostrar publicidad, cumplimos con las regulaciones COPPA.

10. Cambios en esta Política

Si realizamos cambios en esta política de privacidad, actualizaremos esta página y la fecha de "última actualización" que aparece al final.
